# 💜 JourneyPro

Aplicativo mobile para organização de projetos e aprendizado em tecnologia, com foco em UX, acessibilidade e iniciantes na área.

---

## 📌 Sobre o Projeto

O **JourneyPro** é um case de UX/UI desenvolvido para resolver um problema comum entre estudantes iniciantes em tecnologia: a dificuldade de organizar projetos, acompanhar o progresso e manter consistência nos estudos.

Este projeto foi criado como parte de estudos em **UX/UI Design e desenvolvimento de interfaces**, aplicando boas práticas de usabilidade, acessibilidade e organização visual.

---

## 🎯 Problema

Estudantes iniciantes frequentemente enfrentam dificuldades em:

- Organizar projetos de forma clara  
- Acompanhar o progresso de aprendizado  
- Manter consistência nos estudos  
- Visualizar evolução ao longo do tempo  

---

## 💡 Solução

O JourneyPro propõe uma interface simples e intuitiva com:

- 📊 Painel de projetos  
- 📈 Acompanhamento de progresso  
- ✅ Lista de tarefas (checklist)  
- 🧠 Registro de aprendizados  
- 🔗 Integração com ferramentas como Figma e GitHub  

---

## 👩 Persona

**Ana, 22 anos**  
Estudante iniciante em tecnologia que precisa de uma interface clara, acessível e fácil de navegar para organizar seus estudos e projetos.

---

## 🧠 Processo de Design

- Pesquisa e definição de persona  
- Arquitetura de informação  
- Criação de wireframes  
- Definição do fluxo do usuário  
- Desenvolvimento de protótipo de média fidelidade  
- Aplicação de princípios de UX/UI  

---

## 🎨 Decisões de UX

- Uso de **cards** para organização visual  
- Redução de **carga cognitiva**  
- Hierarquia clara de informações  
- Feedback visual em ações do usuário  
- Navegação simples e intuitiva  

---

## ♿ Acessibilidade

- Contraste adequado entre texto e fundo  
- Uso de ícones + texto (sem depender só de cor)  
- Botões com áreas clicáveis maiores  
- Feedback visual (erro, sucesso e carregamento)  
- Baseado em boas práticas de **WCAG**
